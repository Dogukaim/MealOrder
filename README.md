# MealOrder
 Storyboard and Xib UI, MVVM

## Features ...


-Architecture: - [MVVM](https://www.kodeco.com/6733535-ios-mvvm-tutorial-refactoring-from-mvc)

-UI: Xib,Storyboard

-Backend: Firebase

-Networking: Alamofire(Generic)

-3rd party libraries: Kingfisher,JPGProgress,PanModal,
